## Setup
**Dependencies**
- android-tools
- nerdfonts

  You can download from here.
  `https://www.nerdfonts.com/`

```bash
# Clone this repo,and hange this directory.
git clone https://github.com/the-essence-of-life/lazyadb.sh
cd lazyadb.sh

# Then run two commands to init.
chmod +x init.sh
bash init.sh
```
